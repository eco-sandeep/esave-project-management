# Onboarding Checklist for Esave Devs  

## Tools and Actions  
- **MS-365**: Ensure all team members have access to MS-365 resources.  
- **Github**: Familiarize with the repository and workflows.  
- **Heroku**: Set up Heroku accounts and deployment access.  
- **AWS RDS Access**: Grant necessary AWS RDS database access.  
- **Git Usage Documentation**: Review the Git usage documentation for version control practices.
