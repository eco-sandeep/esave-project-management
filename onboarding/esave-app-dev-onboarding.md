# Onboarding Checklist for Esave Devs  

## Tools
- **MS-365**: Ensure all team members have access to MS-365 resources.  
- **Github**: Familiarize with the repository and workflows.  
- **Heroku**: Set up Heroku accounts and deployment access.  
- **AWS RDS Access**: Grant necessary AWS RDS database access.  

## Actions
- **Share Development process documentation**: Review the Git usage documentation for version control practices. Docs here: eco-sandeep/esave-project-management/delivery/development/backlog-to-production.md
- **Secure development guidelines**: Before creating your first PR make sure you have read and understand clearly secure coding guidelines here: EcoCentre-Org/esave/blob/main/coding-guidelines.md
